# 🎓 BCU Website - Birmingham City University Clone

A full-featured ASP.NET Core MVC web application that replicates the Birmingham City University website with a complete admin dashboard for content management.

## ✨ Features

### Public Website
- 🏠 **Dynamic Homepage** with editable content blocks
- 📚 **Course Listings** with search and filtering
- 📰 **News Section** with latest updates
- 🎯 **Promotional Sections** (Clearing, Fast Track, etc.)
- 📱 **Responsive Design** using Bootstrap 5
- 🎨 **Modern UI** matching BCU's design system

### Admin Dashboard
- 📊 **Dashboard Overview** with key metrics
- ✏️ **CRUD Operations** for Courses, News, and Content
- 🖼️ **Image Management** for courses and news
- 📝 **Rich Content Editing** for all text sections
- 🔐 **Secure Admin Area** (ready for authentication)
- 🎯 **Content Block Management** for editable sections

## 🛠️ Technology Stack

### Backend
- **Framework**: ASP.NET Core MVC (.NET 10.0)
- **Database**: SQLite (Entity Framework Core)
- **ORM**: Entity Framework Core 9.0
- **Language**: C# 12

### Frontend
- **CSS Framework**: Bootstrap 5.3
- **Icons**: Font Awesome 6.5
- **Fonts**: Google Fonts (Inter)
- **JavaScript**: Vanilla JS with Bootstrap

### Development Tools
- **IDE**: Visual Studio Code / GitHub Codespaces
- **Version Control**: Git & GitHub
- **Package Manager**: NuGet
